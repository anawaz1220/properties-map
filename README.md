# Arlington Builders - Property Lots Map

An interactive map displaying property lots with detailed information including acreage and dimensions.

## Features

- Interactive lot visualization on satellite and grey basemaps
- Click on any lot to view detailed information
- Smooth zoom and pan animations
- Responsive design for all devices
- Clean, professional UI matching Arlington Builders branding

## Usage

### Embedding in Framer Website

Use the following iframe code to embed the map in your Framer website:

```html
<iframe
    src="https://anawaz1220.github.io/properties-map/"
    width="100%"
    height="600"
    frameborder="0"
    style="border:0;"
    allowfullscreen="">
</iframe>
```

### Direct Link

View the map directly at: `https://anawaz1220.github.io/properties-map/`

## Technologies Used

- Leaflet.js for interactive mapping
- Esri World Imagery for satellite basemap
- CARTO for grey basemap
- GeoJSON for property data
